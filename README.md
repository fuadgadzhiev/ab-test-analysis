# Анализ результатов A/B-теста: изменение стоимости подписки

## Описание проекта
В данном проекте был проведен аналитический разбор результатов A/B-теста. Целью эксперимента было оценить влияние изменения стоимости подписки `premium_no_trial` на ключевые бизнес-показатели: конверсию, ARPU (средний доход на пользователя) и ARPPU (средний доход на платящего пользователя).

## Основные этапы работы
1. **Проверка корректности (A/A-тест):** Сравнение контрольных групп для подтверждения того, что система распределения трафика работает корректно.
2. **Предварительный анализ:** Оценка влияния эксперимента на конверсию и доход.
3. **Очистка данных:** Работа с аномальными значениями («китами») — идентификация и исключение сверхкрупных платежей, которые могли исказить средние показатели.
4. **Статистическая проверка:** Применение статистических методов (`f_oneway`, `chi2_contingency`, `kruskal`) для оценки значимости различий между группами.

## Ключевые выводы
* **Падение конверсии:** Новая цена привела к статистически значимому снижению конверсии (p-value < 0.05).
* **Отсутствие роста выручки:** Статистически значимого роста ARPU после внедрения новой цены не зафиксировано (p-value = 0.9595).
* **Влияние выбросов:** Высокие показатели на сырых данных были обеспечены лишь несколькими аномально крупными транзакциями. После их удаления выяснилось, что средний доход в тестовой группе ниже, чем в контрольной.

**Итоговое бизнес-решение:** Эксперимент признан неудачным. Рекомендуется отказаться от внедрения новой стоимости подписки, так как падение конверсии не компенсируется ростом чека.
